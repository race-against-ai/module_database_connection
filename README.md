## Code Syntax
To test your code type syntax run

```
tox -e type
```

or to check the style syntax run
```
tox -e style
```

to run a lint check, enter
```
tox -e lint_check
```

To update your code when encountering errors on lint check, just run
```
tox -e lint_update
```
